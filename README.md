# School Management System in C++

A simple School Management System project made using C++.

## Features
- Add Student
- View Students
- Search Student
- Delete Student

## Concepts Used
- Object-Oriented Programming (OOP)
- Functions
- Vectors
- Classes and Objects

## How to Run

Compile the program:

```bash
g++ main.cpp -o main
```

Run the program:

```bash
./main
```
